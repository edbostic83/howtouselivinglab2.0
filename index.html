Quick Start (5 minutes)

Use this when you have a tool on your campus and need impact, not more clicks.

  1. Open: Planner on the Living Lab.
  2. Define the habit: In Weekly routine, write one observable behavior (e.g., “Peer feedback every Wednesday”).
  3. Outcome: Add the metric you expect to move in 2–6 weeks (e.g., “+1 rubric level for 60%”).
  4. Owner & cadence: Who owns this and how often? (e.g., “6th ELA team • weekly”).
  5. Export & share: Click Generate Plan → Copy and paste into your PLC notes or email.

Impact Thesis & Core Framework

The Impact Thesis

Impact is a human response to information. Our goal is to move from tool → actionable behavior with transparent evidence.

Three Pillars: Inspire purpose · Motivate action · Build trust with transparent routines.

The Impact Ladder (5 steps)

  1. Tool → What is it?
  2. Use → How is it used weekly?
  3. Behavior → What observable habit changes?
  4. Outcome → What metric moves?
  5. Evidence → What artifact proves it?

Module Guide (What to do in each part of the site)

Module 1 — Planning & Prompting

Tools: Planner · RISEN Prompt Studio

  • Planner: Lock the weekly habit, owner, cadence, outcome, and the standard/policy alignment.
  • RISEN: Build the exact prompt to make AI a thinking partner (Relevance, Intent, Specificity, Execution, Navigation).
Open Planner Open RISEN
Activity 1 (5 min): Name your most expensive/complex tool. In Planner, define the Behavior and Outcome it should drive. In RISEN, generate the coaching prompt you’ll use with staff.

Module 2 — Data & Discovery

Tools: Data Prep Coach · AI Platform Finder

  • Data Prep Coach: Write the AI‑ready spec (grain, window, fields, privacy). If data is messy, fix the process before using AI.
  • Platform Finder: Choose by need (create/transform/analyze/communicate/automate), inputs, outputs, constraints, and risk.
Open Data Prep Open Platform Finder
Activity 2 (5 min): In Platform Finder, choose a Goal (e.g., Automate) and an Input (e.g., Spreadsheet). Copy the top two Search Strings it generates.

Module 3 — Communication & Integration

Tools: 3‑Sentence Story · Process Plug‑In · Sell It

  • 3‑Sentence Story: Challenge → Simple Action → Human Impact.
  • Process Plug‑In: Add one AI‑ready step into an existing, required process (IEP, 504, PD, hiring).
  • Sell It: Turn your plan into a two‑sentence pitch for principals/board.
Open 3‑Sentence Story Open Process Plug‑In Open Sell It
Activity 3 (3 min): Using your Ladder and Planner, write a two‑sentence pitch in Sell It. Share with your table and refine once.
